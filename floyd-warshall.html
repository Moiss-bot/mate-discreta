Formulación matemática

Recurrencia de Floyd-Warshall

Para cada vértice intermedio k, el algoritmo evalúa si la ruta i → k → j mejora la distancia conocida d[i][j]. La recurrencia d[i][j] = min(d[i][j], d[i][k] + d[k][j]) se aplica iterativamente para k = 1, 2, …, |V|, produciendo la matriz de distancias mínimas entre todos los pares.

1

Construir la matriz de adyacencia con distancias directas e ∞ para pares no conectados.

2

Para cada k ∈ V, evaluar si pasar por k mejora d[i][j].

3

Obtener la matriz final con caminos mínimos entre todos los pares ordenados.
